Herramientas

DeepSeek es un modelo de IA avanzado para procesamiento de lenguaje. Figma es una herramienta de diseño colaborativo en la nube para interfaces y prototipos. Visual Studio Code es un editor de código ligero y personalizable con soporte para múltiples lenguajes y extensiones. GitHub es una plataforma de desarrollo basada en Git para alojar, revisar y colaborar en proyectos de software. Juntas, estas herramientas facilitan el desarrollo, diseño y gestión de proyectos tecnológicos de manera eficiente y colaborativa.

Estats

Java es un lenguaje de programación orientado a objetos, robusto y multiplataforma, usado en aplicaciones empresariales y Android. HTML es el lenguaje base para estructurar contenido en la web, definiendo la jerarquía de elementos en una página. CSS complementa a HTML, permitiendo dar estilo y diseño a los sitios web con colores, layouts y animaciones. SQL es un lenguaje especializado para gestionar y consultar bases de datos relacionales, permitiendo manipular datos eficientemente. Juntos, estos lenguajes son fundamentales para el desarrollo web, aplicaciones y manejo de datos.
